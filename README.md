esse é um projeto ainda em andamento, trata-se de um conversor de moedas, capaz de utilizar dados reais para calcular o valor do real quando convertido para outras moedas.
Tecnologias usadas: Java, APIs, Jackson, MYSQL
objetivos alcançados: conexão com banco de dados; Adicionar valores ao banco; puxar valores reais via API do BACEN, Leitura de .JSON
próximos objetivos: Criar calculadora em C++; Conectar o Código em java ao C++; adicionar suporte para mais moedas;
objetivos gerais do projeto: desenvolver as habilidades de integração entre multiplas linguagens, bancos de dados e APIs e as habilidades relacionadas à estruturação em projetos reais, principalmente no back end.
