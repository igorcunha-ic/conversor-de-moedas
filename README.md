esse é um projeto pre-concluido, trata-se de um conversor de moedas, capaz de utilizar dados reais para calcular o valor do real quando convertido para outras moedas.
Tecnologias usadas: Java, C++, APIs, Jackson, MYSQL, CMake, SpringBoot
objetivos alcançados: conexão com banco de dados; Adicionar valores ao banco; puxar valores reais via API do BACEN, Leitura de .JSON, conectar o código em java ao C++, adicionar suporte para mais moedas, funcionamento correto da calculadora em C++
próximos objetivos: front end;
objetivos gerais do projeto: desenvolver as habilidades de integração entre multiplas linguagens, bancos de dados e APIs e as habilidades relacionadas à estruturação em projetos reais, principalmente no back end.
